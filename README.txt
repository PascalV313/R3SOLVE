*** System & Hardware Requirements & Installation:***
-------------------------------------------------

Unzip project files and save in desired location then run the solution file with Visual Studio 17.

Locate SQL BACKUP folder and restore the sql database file ServicesDB.bak to the server management studio and run the application.(support link below if necessary)


If using directive not installed, include the Using Directive: using Dapper;

locate the connection string from Server Explorer, right click on servicesDb and choose properties, copy & replace the connection string.

Validations, ToolTips, Help Menu, Shortcuts and Tab Orders are all included in the application is 

Jing SWF files can be viewed using Internet explorer.
